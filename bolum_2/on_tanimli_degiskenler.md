```ruby
puts global_variables
```

# Pseudo (Gerçek Olmayan) Değişkenler
**Değişken** (_Variable_) gibi görünen ama **Sabit** (_Constant_) gibi davranan ve kesinlikle **değer ataması yapılamayan** şeylerdir.

| Değişken | Açıklama |
| -- | -- |
| self | -- |
| nil | -- |
| true | -- |
| false | -- |
| \_\_FILE\_\_ | -- |
| \_\_LINE\_\_ | -- |
